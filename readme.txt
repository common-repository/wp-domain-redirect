WP Domain Redirect plugin is intended for domain redirection based on country. 

== Description ==

WP Domain Redirect plugin is used to redirect your domain based on country. It will catch the IP address of the system and identify the country and it will redirect to the domain intended for that country.

You can add domains based on your country
Please visit http://www.dmitritech.com to download the plugin.

== Installation ==

1. Upload the files to the /wp-content/plugins/wp-domain-redirect/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Domain Redirect menu in WordPress.By selecting the submenu 'Add New', you can add the domain name based on the country.
See the Screenshots for for more details.
